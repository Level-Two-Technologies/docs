title: "Get the credit report"
api: "GET https://api.ltwo.io/api/v1/report"
description: "This endpoint updates an existing user."
---

### Query Parameters

<ParamField query="report_id" type="string" required>
  Report id generated while ordering the report
</ParamField>

<ParamField query="token_id" type="string" required>
  Unique token to retrieve the report.
</ParamField>

### Response

Generates and delivers the credit report that you previously ordered through the Order Credit Report call.

### Upcoming changes

## Report Formats and Delivery Methods
The report can be generated in a number of different formats: HTML, XML, raw, and so on. It can be delivered in one of two ways:
1. In the body of the response.
2. As an argument to a callback function that's invoked if the report generation is successful (HTTP status 200). To use this delivery method, you must implement a JSONP callback function that takes a single string argument, and pass the name of your function to the onSuccess query parameter.


<RequestExample>
```bash cURL
curl --location --request GET 'https://api.ltwo.io/api/v1/report?report_id=a007ace7-becd-44d1-8b5e-2c07ee3226e5&token_id=539c7cb6-f416-4c9d-9cae-ebae2a286873' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>'
```
</RequestExample>
