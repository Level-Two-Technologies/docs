---
title: "Refresh the token"
api: "PUT https://api.ltwo.io/api/v1/income/summary"
description: "This endpoint refreshes the token for getting summary"
---

### Body

<ParamField body="report_id" type="string" required>
    Report id generated while ordering the report
</ParamField>

<ParamField body="user_id" type="string" required>
    User id used while ordering the report
</ParamField>

### Response

<ResponseField name="report_id" type="string">
    Unique id of the report ordered
</ResponseField>

<ResponseField name="token_id" type="string">
    Unique token to retrieve the report. The token is valid for 30 min only
</ResponseField>

<RequestExample>
    ```bash cURL
    curl --location --request PUT 'https://api.ltwo.io/api/v1/income/summary' \
    --header 'Content-Type: application/json' \
    --header 'Authorization: Token <token>' \
    --data-raw '{
        "user_id": "d3ebbd44-470a-43c5-ab65-a4ed34ef6824",
        "report_id": "d3ebbd44-470a-43c5-ab65-a4ed34ef6824"
    }'
    ```
</RequestExample>

<ResponseExample>

    ```json Response
    {
        "report_id": "a007ace7-becd-44d1-8b5e-2c07ee3226e5",
        "token_id": "539c7cb6-f416-4c9d-9cae-ebae2a286873"
    }
    ```

</ResponseExample>
