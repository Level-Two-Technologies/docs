---
title: "Get the income summary"
api: "GET https://api.ltwo.io/api/v1/income/summary"
description: "This endpoint retrieves the income summary."
---

### Query Parameters

<ParamField query="report_id" type="string" required>
    Report id generated while ordering the report
</ParamField>

<ParamField query="token_id" type="string" required>
    Unique token to retrieve the report.
</ParamField>

### Response

Delivers the income summary that was previously ordered through the Order Income Summary call.

<RequestExample>
```bash cURL
curl --location --request GET 'https://api.ltwo.io/api/v1/income/summary?report_id=a007ace7-becd-44d1-8b5e-2c07ee3226e5&token_id=539c7cb6-f416-4c9d-9cae-ebae2a286873' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>'
```
</RequestExample>
